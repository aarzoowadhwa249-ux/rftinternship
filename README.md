#  Reverse String in Java

This is a basic Java program that reverses a string using a loop.

##  Description
The program takes a string ("Hello World") and reverses it by iterating from the last character to the first.

##  Features
- Simple and easy to understand
- Beginner-friendly Java program
- Demonstrates string manipulation using loops

##  How It Works
- Store the original string
- Loop through the string from end to start
- Append each character to form the reversed string

##  Output
Original String: Hello World  
Reversed String: dlroW olleH
